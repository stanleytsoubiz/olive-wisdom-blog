## 附錄：執行注意事項

### 防重複（防 Cannibalization）提醒
以下幾組選題與現有文章邊界需要在執行前確認：

1. **「橄欖油炒菜會致癌嗎」** vs 現有「炒菜發煙點」「加熱穩定性」文章：新文主打「致癌疑慮破解」，現有文保留技術說明；新文需內鏈至舊文，H1 不得與舊文 H1 重複。
2. **「空腹喝橄欖油」** vs 現有「晨間儀式代謝」文章：確認兩文主關鍵字不重疊；「晨間儀式」定位生活風格，「空腹喝」定位健康問答，邊界清晰。
3. **「橄欖油失智症」** vs 現有「羥基酪醇腦健康」「SIRT1 長壽」文章：新文聚焦「失智/阿茲海默」疾病關鍵字，舊文保留化合物層面；新文需引用舊文作為科學來源。
4. **「橄欖油糖尿病」** vs 現有「心血管功效」「每日攝取量」：糖尿病是獨立受眾群，邊界清晰，可推進。
5. **「產地比較」** vs 現有「Terroir 產地風土」：確認舊文是否已精準命中「西班牙 vs 義大利 vs 希臘」的比較意圖；若是，新文應從不同角度切入（如「購買選擇視角」），或更新強化舊文而非新建。

### 競品機會摘要
- **mybest / iGood** 類「排行榜推薦」文：商業意圖強，但公正性可疑。本站若能建立「評測標準透明、引用多酚數據、不收業配」的選購指南，可建立差異化權威。
- **品牌部落格（澳翠、橄欖先生等）**：流量強但利益衝突明顯，讀者有不信任感。本站中立立場是核心競爭優勢。
- **媒體健康版（ETtoday、TVBS）**：聳動標題多、深度不足，回訪率低。本站長篇科學文可建立更高的 SERP 權威信號。
