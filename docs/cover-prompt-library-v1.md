## 使用說明

每個類別提供 **A / B 兩個備選 prompt**，差異在於：
- **A 版**：靜物近景為主，強調質感與氛圍
- **B 版**：場景感稍強，具備更多環境敘事線索

G1 確認選題後，每篇文章選用對應類別的 A 或 B，由 D1 呼叫 Imagen 4 生成。
